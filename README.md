## A city guide Vue.js / Express.js web app.

This is a simple Vue.js / Express.js city guide web app for **ExtensionEngine Summer Camp 2019**.



### How to run the app

In `server/src/config.js` fill in your database user, password, dialect.

### Client - Terminal A

```
cd client
npm install
npm run serve
```

### Server - Terminal B

```
cd server
npm install
npm run seed
npm run start
```
